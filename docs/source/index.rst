Welcome to Manim Physics's documentation!
=========================================
``manim-physics`` is a 2D physics simulation `manim <https://docs.manim.community/en/stable/>`_ 
plugin that allows you to generate complicated scenes in various branches of
Physics such as rigid mechanics, electromagnetism, wave etc.
